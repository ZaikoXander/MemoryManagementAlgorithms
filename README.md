### Memory Management Algorithms:

#### How can I run this code?

The requirements to run the code are:

- Install `nodejs` in your computer
- Install yarn by running `npm install --global yarn`
- Run `yarn` to install the dependencies
- Run `yarn start` to initialize the application

#### How can I contribute?

If you wish to contribute you may:

- Open an issue
- Fork the repository and then create a pull requests.
